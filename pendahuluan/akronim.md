# Akronim dan definisi

| Akronim                                     | Definisi                                                                                                                                                                                                                                                           |
| ------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Joomla!                                     | Joomla! – Sistem pengurusan kandungan (CMS) sumber terbuka yang popular digunakan untuk membina laman web dan aplikasi dalam talian.                                                                                                                               |
| Search Engine Optimization (SEO)            | Amalan mengoptimumkan kata kunci laman web untuk meningkatkan kedudukan laman web dalam hasil enjin carian.                                                                                                                                                        |
| Meta Description                            | Ringkasan kandungan halaman web yang muncul dalam hasil enjin carian untuk membantu pengguna memahami halaman sebelum masuk.                                                                                                                                       |
| Meta Title                                  | Tag tajuk halaman web yang muncul dalam hasil enjin carian dan tab penyemak imbas dan ianya penting untuk SEO.                                                                                                                                                     |
| Sitemap                                     | Fail yang menyenaraikan semua halaman di laman web bagi membantu enjin carian membuat indeks laman web dengan berkesan.                                                                                                                                            |
| User Experience (UX)                        | Pengalaman keseluruhan yang dialami pengguna semasa berinteraksi dengan laman web atau aplikasi, termasuk kebolehgunaan dan kebolehcapaian.                                                                                                                        |
| Content Management System (CMS)             | Perisian yang membolehkan pengguna mencipta, mengurus dan mengubah suai kandungan digital tanpa memerlukan pengetahuan teknikal yang luas.                                                                                                                         |
| Secure Sockets Layer (SSL)                  | Protokol keselamatan yang menyulitkan data (data encryption) yang dihantar antara pelayar web dan pelayan serta melindungi maklumat sensitif.                                                                                                                      |
| Webserver                                   | Sistem perisian atau perkakasan yang menjadi hos laman web dan menyajikan kandungan web kepada pengguna melalui internet.                                                                                                                                          |
| MySQL                                       | Sistem pengurusan pangkalan data hubungan (RDBMS) sumber terbuka yang biasa digunakan untuk aplikasi web.                                                                                                                                                          |
| PhpMyAdmin                                  | Sistem berasaskan web yang membolehkan pengguna mengurus pangkalan data MySQL dengan antara muka pengguna.                                                                                                                                                         |
| URL                                         | Alamat web yang digunakan untuk mengakses halaman web atau sumber tertentu di internet.  Ia terdiri daripada protokol (seperti https://), nama domain (uitm.edu.my), dan laluan direktori (/halaman).                                                              |
| Hypertext Transfer Protocol Secure (HTTPS)  | Versi HTTP yang selamat yang menyulitkan (encrypt) data antara pelayar pengguna dan laman web bagi memastikan kerahsiaan dan perlindungan daripada ancaman siber.  Ia biasanya digunakan untuk transaksi dalam talian, halaman log masuk, dan komunikasi sensitif. |
| Web Content Accessibility Guidelines (WCAG) | Merupakan satu set garis panduan, yang dibangunkan oleh Konsortium Web Sedunia (W3C), yang menjelaskan cara menjadikan kandungan web lebih mudah diakses oleh orang kurang upaya.                                                                                  |
