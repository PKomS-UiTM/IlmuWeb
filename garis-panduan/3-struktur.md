# 3.0 Struktur Laman Web

Struktur laman web yang teratur dan sistematik amat penting bagi memastikan kejelasan, kebolehgunaan serta kecekapan penyampaian maklumat di seluruh laman web UiTM. Struktur yang diseragamkan membantu mewujudkan pengalaman pengguna yang konsisten dan mudah diakses, sambil menyesuaikan keperluan khusus bagi laman web fakulti, kampus cawangan dan pejabat. Bagi memastikan keseragaman dan navigasi yang jelas, struktur laman web UiTM disyorkan untuk dibahagikan kepada dua menu utama, iaitu:

#### Menu Atas (_Top Menu_)

Menu atas berfungsi sebagai akses pantas kepada pautan dan ciri utama yang sering digunakan oleh pelawat laman web. Antara elemen yang boleh dimasukkan termasuk:

- Permohonan (_Apply Now_)
- Berita (News)
- Acara (Event)
- Pautan penting lain yang relevan dengan fungsi laman web

Menu atas ini bertindak sebagai penunjuk navigasi utama, membantu pengguna mengenal pasti lokasi mereka dalam laman web serta mengakses bahagian penting dengan cepat dan mudah.

#### Menu Utama (_Main Menu_)

Menu utama menyediakan navigasi yang lebih mendalam dan menyeluruh, mencerminkan kandungan teras laman web berkenaan. Menu ini biasanya merangkumi pautan ke kategori utama dan sumber maklumat yang berkaitan dengan peranan serta fungsi laman web tersebut. Sebagai contoh, bagi laman web fakulti, menu utama boleh merangkumi:

- Profil Fakulti / Kampus / Pejabat
- Program Akademik
- Penyelidikan dan Inovasi
- Perkhidmatan dan Kemudahan
- Maklumat Staf
- Hubungi Kami

Struktur dua peringkat menu ini mewujudkan hierarki navigasi yang jelas, seterusnya meningkatkan pengalaman pengguna dengan memudahkan capaian kepada maklumat yang diperlukan. Pendekatan ini juga membantu mengekalkan keseragaman reka bentuk dan fungsi di seluruh laman web UiTM, sekaligus memperkukuh imej profesional serta kehadiran digital universiti secara menyeluruh.

## 3.1 Laman Web Utama UiTM

Laman web utama UiTM berfungsi sebagai hab digital utama universiti, menyediakan maklumat penting kepada pelajar, pensyarah, staf, alumni serta masyarakat umum. Struktur ini direka bentuk untuk memastikan navigasi yang intuitif, reka bentuk yang seragam dengan identiti korporat UiTM, dan kandungan yang tersusun bagi meningkatkan aksesibiliti serta pengalaman pengguna.

Sebagai laman rasmi utama universiti, laman ini mesti memaparkan pautan yang jelas kepada program akademik, kemasukan pelajar, penyelidikan dan inovasi, berita semasa, serta sumber pentadbiran penting. ## 3.4 Laman Web Pejabat / Pusat / Bahagian

Laman web bagi Pejabat, Pusat atau Bahagian UiTM berfungsi sebagai saluran komunikasi rasmi untuk menyampaikan maklumat berkaitan fungsi, perkhidmatan, program serta inisiatif yang dijalankan oleh unit berkenaan. Laman web ini menjadi sumber rujukan utama bagi warga UiTM dan pihak luar dalam mendapatkan maklumat mengenai tanggungjawab, struktur organisasi, serta kemudahan yang ditawarkan oleh pejabat atau pusat tersebut.

Struktur laman web ini direka untuk memastikan penyampaian maklumat yang teratur, mesra pengguna, serta sejajar dengan identiti korporat UiTM. Setiap laman web perlu mengekalkan konsistensi dari segi reka bentuk, navigasi, dan gaya penyampaian agar pengalaman pelawat kekal seragam di seluruh rangkaian laman web UiTM.
