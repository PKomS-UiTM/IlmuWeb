# 6.0 Keperluan Teknikal

Bagi memastikan operasi laman web UiTM berjalan lancar, selamat dan mudah diakses, semua platform UiTM hendaklah mematuhi keperluan teknikal yang telah diseragamkan. Garis panduan ini meliputi penggunaan Sistem Pengurusan Kandungan (CMS), piawaian pelayan (hosting), amalan terbaik SEO, serta integrasi media sosial bagi membentuk kehadiran digital UiTM yang seragam dan berkesan.

---

## 6.1 Sistem Pengurusan Kandungan (CMS)

Universiti Teknologi MARA (UiTM) menetapkan bahawa semua laman web rasmi UiTM hendaklah dibangunkan dan diuruskan menggunakan Sistem Pengurusan Kandungan (Content Management System, CMS) Joomla!. Penggunaan CMS ini adalah standard rasmi universiti bagi memastikan keseragaman reka bentuk, keselamatan, penyelenggaraan dan kemudahan integrasi di seluruh rangkaian laman web UiTM.

Sebarang penggunaan atau pertukaran kepada CMS lain (seperti WordPress atau mana-mana platform lain) adalah tidak dibenarkan tanpa kebenaran bertulis daripada Pejabat Komunikasi Strategik UiTM. Pematuhan kepada garis panduan ini adalah wajib bagi semua entiti UiTM termasuk fakulti, kampus cawangan, pejabat, pusat, akademi, institut dan unit di bawah universiti.

---

## 6.2 Infrastruktur Pelayan dan Pengurusan Domain

Perkhidmatan pelayan laman web rasmi UiTM disediakan oleh Jabatan Digital UiTM. Pegawai Perhubungan Awam serta webmaster yang dilantik akan diberikan akses cPanel, bagi membolehkan pengurusan laman web dilakukan dengan lebih efisien termasuk pengurusan fail, pangkalan data, dan tetapan keselamatan.

Bagi mengekalkan keseragaman dan profesionalisme kewujudan laman web UiTM dalam talian, Pejabat Komunikasi Strategik telah menetapkan piawaian penamaan subdomain UiTM. Semua laman web rasmi hendaklah menggunakan subdomain di bawah domain utama UiTM, contohnya:

- fakulti.uitm.edu.my
- pejabat.uitm.edu.my
- cawangan.uitm.edu.my

Domain yang bertujuan untuk kegunaan konferensi, individu, atau aktiviti sementara tidak dibenarkan menggunakan subdomain UiTM. Semua laman web hendaklah mempunyai justifikasi rasmi serta mendapat kelulusan daripada Pejabat Komunikasi Strategik sebelum pendaftaran subdomain dilakukan.

Sebagai alternatif, laman web untuk konferensi, seminar, bengkel, pertandingan, atau projek sementara disarankan untuk dibangunkan menggunakan domain khas bukan rasmi, contohnya:

- menggunakan domain umum seperti .my, .com, atau .org
- menggunakan subfolder atau microsite di bawah laman web rasmi, contohnya:
  - fakulti.uitm.edu.my/konferensi
  - cawangan.uitm.edu.my/seminar-digital

Pendekatan ini membantu memastikan subdomain rasmi UiTM kekal teratur, selamat, dan berasaskan entiti kekal universiti. Proses pendaftaran dan penyelenggaraan domain diurus secara berpusat oleh Jabatan Digital UiTM dengan kerjasama Pejabat Komunikasi Strategik, bagi memastikan pematuhan kepada keperluan, dasar keselamatan, dan garis panduan universiti.

## 6.3 Pengoptimuman Enjin Carian (SEO)

Pengoptimuman Enjin Carian (SEO) berperanan penting dalam memastikan laman web UiTM mudah ditemui oleh pelajar, warga UiTM, dan masyarakat umum melalui enjin carian. Semua laman web UiTM perlu mengamalkan prinsip SEO bagi meningkatkan keboleh lihatan, keboleh capaian dan index yang baik di enjin carian. Amalan terbaik SEO adalah:

- Menggunakan tajuk meta (meta title) bagi setiap halaman atau kandungan.
- Penyediaan deskripsi meta (meta description) yang jelas dan menarik bagi meningkatkan paparan di enjin carian.
- Memasang/membolehkan Search Engine Friendly (SEF) URL.
- Reka bentuk responsif (responsive design) bagi sokongan disemua jenis peranti mudah alih.
- Optimasi kelajuan laman dengan mengurangkan masa muat turun melalui pemampatan imej, skrip dan penggunaan caching.

## 6.4 Integrasi Media Sosial dan Platform Luaran

Integrasi media sosial membantu meningkatkan interaksi, keterlibatan dan penyampaian maklumat antara UiTM dengan pelajar, staf serta masyarakat umum. Panduan integrasi media sosial pada laman web UiTM Adalah seperti berikut:

- Sertakan ikon pautan ke akaun media sosial rasmi UiTM/fakulti/pejabat/cawangan (seperti Facebook, Instagram, X, TikTok, LinkedIn dan platform lain yang relevan).
- Pastikan identiti dan penjenamaan (branding) adalah konsisten di semua saluran media sosial.
- Patuhi dasar keselamatan data dan privasi UiTM semasa menghubungkan laman ke platform pihak ketiga.
- Semak dan kemas kini semua pautan media sosial secara berkala bagi mengelakkan pautan rosak atau tidak sah.
- Sebarang KMP tambahan untuk strim kandungan media sosial ke laman web adalah tidak dibenarkan.
